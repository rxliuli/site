# SelectText

SelectText is a Chrome extension that makes text extraction effortless. Simply click the extension icon, drag to select any region on a webpage, and let OCR technology extract the text for you.

## Features

- **Click-and-drag Selection**: Intuitive region selection on any webpage
- **OCR Text Extraction**: Powered by Chrome Lens for accurate text recognition
- **One-click Copy**: Instantly copy recognized text to clipboard
- **Dark Mode**: Comfortable viewing in any lighting condition
- **Fast & Efficient**: Quick processing with minimal overhead

## How It Works

1. Click the SelectText extension icon in your browser
2. Drag to select the text area you want to extract
3. Wait for OCR processing (usually just seconds)
4. Click on text blocks to copy them to clipboard
5. Press `Esc` to exit selection mode

## Technology

Built with modern web technologies:

- **WXT Framework**: Modern web extension development
- **TypeScript**: Type-safe development
- **Chrome Lens OCR**: High-quality text recognition
- **Tesseract.js**: Fallback OCR engine

## Use Cases

- Extract text from images
- Copy text from PDFs rendered in browser
- Capture text from videos or screenshots
- Get text from non-selectable content
