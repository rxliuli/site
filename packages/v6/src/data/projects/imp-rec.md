## Features

- Menu bar only — no dock icon, negligible idle footprint
- One-click recording with system window/display picker
- Built-in trim editor with frame-stepping (arrow keys, comma/period)
- Exports to MP4 via ffmpeg x264 for small file sizes
- Falls back to AVFoundation if ffmpeg is not installed

## Requirements

- macOS 14.0 (Sonoma) or later
- Screen recording permission (prompted on first launch)
