# Scrub - Clear Site Data

## Introduction

One-click clear all site data (cookies, storage, cache) for the current website. Originally built for iOS Safari — where clearing site data requires digging through system settings — then expanded to all major browsers.

## Key Features

- 🧹 One-click to clear all site data for the current tab
- 🍪 Clears cookies, localStorage, sessionStorage, IndexedDB, Cache Storage, and OPFS
- 📱 Built for iOS Safari first, now available on Chrome, Firefox, and Safari (macOS/iOS)
- 🔒 No data collection — everything runs locally in your browser
